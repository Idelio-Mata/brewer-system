## Descrição do Projeto

Sistema web desenvolvido em **Java 17** utilizando **Spring MVC com configuração manual (sem Spring Boot)** com o objetivo de estudo e domínio da arquitetura do framework.

A aplicação será empacotada como **WAR** e executada em servidores **Servlet Containers**.

## Arquitetura

O projeto segue o padrão **MVC (Model–View–Controller)**.

Características principais:

- Java Web tradicional
- Arquitetura baseada em **Spring MVC**
- Renderização no servidor (Server Side Rendering)
- Utilização de **Template Engine**
